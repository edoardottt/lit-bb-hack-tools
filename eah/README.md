# eah

**e**xtract **a**ll **h**osts

Take as input on stdin a list of urls and print on stdout all the hosts sorted. 

### Install

- `git clone https://github.com/edoardottt/lit-bb-hack-tools`
- `cd lit-bb-hack-tools/eah`
- `go build -o eah`
- `sudo cp eah /usr/bin`

### Usage

`cat urls | eah`

### Sample output

```
[ 1148 ] www.google.com
[ 934 ] linux.com
[ 73 ] store.microsoft.com
[ 21 ] hackerone.com
```
